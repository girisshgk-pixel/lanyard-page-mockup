What this document covers

Overview

This note covers the implementation of a reusable programmatic page template for category-level landing pages on badge-a-minit.com.au. Each page is driven by ACF custom fields so the SEO team can populate and update content without touching code.

The SEO Content Spec column shows the content design and variable pattern. The WordPress / ACF Implementation column shows the PHP template code for each section.

H1 + Lead Paragraph

Trust Signals

Trust Bar

Placement: directly under the lead text or above the H1. Provides immediate social proof and creates a quotable fact for AI scrapers (ChatGPT, Perplexity, Google SGE).

SEO Content Spec
Trust bar copy (default):

"Trusted by 5,000+ Australian Schools, Clubs, & Government Departments"

SEO Note: This line is intentionally written as a quotable fact so AI scrapers can extract and surface it as a trust signal when recommending suppliers — phrasing must remain assertive and numerical.
WordPress / ACF Implementation
ACF Field: trust_bar_text (Text, pre-filled default)

<div class="trust-bar">
  <p><?php echo esc_html(
    get_field('trust_bar_text')
  ); ?></p>

SEO Implementation

Dynamic Meta Tags

Add to functions.php or use Rank Math custom field variables. Do not use both simultaneously — they will conflict.

<?php
// Dynamic title tag
add_filter('pre_get_document_title', function($title) {
    if (is_page_template('template-category-page.php')) {
        $category_name = get_field('category_name');
        return $category_name . ' Australia | Trusted Since 1984';
    }
    return $title;
});

// Dynamic meta description
add_action('wp_head', function() {
    if (is_page_template('template-category-page.php')) {
        $category_name = get_field('category_name');
        $product_term  = get_field('category_product_term');
        echo '<meta name="description" content="Shop premium '
          . esc_attr($category_name)
          . ' Australia-wide. Trusted supplier of '
          . esc_attr($product_term)
          . ' since 1984. Fast shipping. Wholesale pricing available.">';
    }
});
ℹ️
Rank Math NoteIf using Rank Math, map ACF fields via Rank Math's custom variable settings to avoid conflicts with the wp_head hook above. Do not use both methods simultaneously.

URL Architecture

URL Structure

All category pages must follow this slug pattern. Slugs are set manually by the SEO team — no automation required for this step.

domain.com.au/[category-slug]-australia/

Examples:

/lanyards-australia/
/lapel-pins-australia/
/id-cards-australia/
🔗
Canonical noteEnsure each category page has a self-referencing canonical tag. If using Rank Math, this is set automatically. If manual, add <link rel="canonical" href="[full page URL]"> in the page template.
9
Quality Compliance

Thin Content Rule

⚠️
Required — add this instruction inside the ACF WYSIWYG field description for below_fold_content "Required. Minimum 300 words. Include FAQs, technical specs, or a material guide specific to this category. Pages without this content will be flagged for review by the SEO team."

This is a Google Thin Content compliance requirement. Every programmatic page must have at least 300–500 words of unique, category-specific text below the fold. Templated pages that do not meet this threshold risk being de-prioritised in indexing.
